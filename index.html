Optimisez un site web existant

L'objectif de ce projet est l'optimisation du site web de l'agence "La Panthère", une entreprise de web design qui souhaite améliorer son référencement naturel sur Internet.

Diverses optimisations ont été apportées en termes de SEO, mais également de performances et d'accessibilité.

Les outils employés pour mesurer le gain de ces optimisations sont Google Lighthouse et Wave. Ces outils sont disponibles via des extensions pour navigateurs.

Il est ainsi possible de comparer leurs résultats, entre la version originale et la version optimisée du site de l'agence La Panthère.
